## Getting Started

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

Then open your browser to `http://localhost:8080`

### Alternative: Open Directly

You can also open `index.html` directly in your browser since the app uses CDN-hosted axios.

## Technologies

- **Axios**: HTTP client for API requests
- **JSONPlaceholder**: Free fake REST API for testing
- **Vanilla JavaScript**: No frameworks needed
- **Modern CSS**: Gradient backgrounds and smooth animations

## Usage

Click any of the three buttons to fetch different data:
- **Fetch Users**: Get a list of users with their details
- **Fetch Posts**: View recent blog posts
- **Fetch Todos**: See todo items with completion status

The app demonstrates:
- GET requests with axios
- Query parameters
- Error handling
- Request/response interceptors
- Async/await syntax
